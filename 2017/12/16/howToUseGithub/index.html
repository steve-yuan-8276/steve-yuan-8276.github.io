工作流

你的本地仓库由 git 维护的三棵“树”组成：

  • 第一个是你的工作目录，它持有实际文件；
  • 第二个是 缓存区（Index），它像个缓存区域，临时保存你的改动；
  • 第三个是 HEAD，指向你最近一次提交后的结果。

有用的贴士

  • 内建的图形化 git：gitk
  • 彩色的 git 输出：git config color.ui true
  • 显示历史记录时，只显示一行注释信息：git config format.pretty oneline
  • 交互地添加文件至缓存区：git add -i
