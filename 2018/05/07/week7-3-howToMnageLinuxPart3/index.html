Linux网络相关

Linux上，常用的网络查询工具集有两个，分别是net-tools和iproute。两者在功能能有重叠，后者iproute是前者net-tools的替代方案：

用途net-tool（替代）iproute2
地址和链路配置ifconfigip addr, ip link
路由表routeip route
邻居arpip neigh
VLANvconfigip link
隧道iptunnelip tunnel
组播ipmaddrip maddr
统计netstatss

iptables [-t 表] [-命令] [匹配] [操作]

iptables -t nat -nvl

注：
-t 指定表；如果不加-t，默认显示filter表相关信息；
-n 表示不针对ip反解析主机名；
-L 表示列出；
-v 表示列出详细信息；
-nvL 表示查看该表格的详细规则；

iptables -F //删除所有规则

iptables -Z //包及流量计数器归零

iptables -I INPUT -s 8.8.8.8 -j DROP //插入一条规则，将所有来自8.8.8.8的数据包丢弃

iptables -D INPUT -s 8.8.8.8 -j DROP //删除上一条规则，注意删除规则时必须和插入规则保持一致

iptables -A INPUT -s 1.1.1.1 -p tcp –dport 80 -j DROP //插入一条规则，将来自2.2.2.2的tcp协议，发送到本机80端口的数据包丢弃

iptables -A INPUT -s 192.168.1.0/24 -i ens33 //增加一条规则，将来自192.168.1.0/24这个网段，且作用在ens33上的包放行

iptables -D [链名] [规则序号]

iptables -nvL –line-numbers
