Homebrew 包管理

Installation
homebrew常用命令：
命令作用
brew install <软件名>安装软件
brew uninstall <软件名>卸载软件
brew search <软件名>查询软件
brew list列出已安装的软件
brew update更新brew
brew home用浏览器打开brew的官方网站
brew info <软件名>显示软件信息
brew deps显示包依赖
