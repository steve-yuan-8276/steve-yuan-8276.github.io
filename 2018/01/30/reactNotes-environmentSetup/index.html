命令 作用
brew install <软件名> 安装软件

brew search <软件名> 查询软件
brew list 列出已安装的软件
brew update 更新brew
brew home 用浏览器打开brew的官方网站
brew info <软件名> 显示软件信息
brew deps 显示包依赖
